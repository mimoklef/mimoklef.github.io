Moocs

I made this project for my first intership in summer 2018.

It was my first real web application and it took me a little bit less than one month to learn PHP for the first time and to create the app.

A few features from this app (The last one is my favorite)


User previlieges

It has two account types, admins and members.
Admins can access some pages that members can't reach.
As you can see in the pictures here, the nav bar is different
depending on the account type.

And of course in those pages the first PHP statement is a condition that verifies user previlieges and displays a message if the user isn't a admin.

Profiles

- All users have a profile page that everyone can visit. This page contains a search form to search for users by their usernames.
- Users can also upload a picture to use as profile picture.

Mailbox


Users can send mails to each others and access their mailboxes.

Member counter

In the footer. Yeah like in the old forums.
